Pixel Office

Real-time pixel-art visualization for AI coding assistants.
Works with both OpenCode and Claude Code.

GitHub Install

Screenshots

Features

  • Dual Support - Works with OpenCode and Claude Code simultaneously
  • Tabbed Interface - Switch between OpenCode Office and Claude Office
  • Live Visualization - See agents move, work, and communicate in real-time
  • Session Tracking - Monitor multiple sessions across different repos
  • Mobile Ready - Connect via local network + QR code

Installation

  1. Install globally: npm install -g opencode-pixel-office
  2. Run installer: opencode-pixel-office install
  3. Start server: opencode-pixel-office start

Dashboard opens at http://localhost:5100

CLI Commands

  • opencode-pixel-office install - Install for OpenCode + Claude Code
  • opencode-pixel-office start - Start server & open browser
  • opencode-pixel-office stop - Stop the server
  • opencode-pixel-office status - Check installation status
  • opencode-pixel-office uninstall - Remove everything
