OpenCode Pixel Office

Real-time OpenCode agent visualization in a pixel office scene.

Installation Features

Screenshots

Features

  • Auto-starts the local Pixel Office server
  • Streams OpenCode events in real time
  • Agent movement, messages, and status indicators
  • Session and todo panels for at-a-glance context
  • Mobile Ready: Connect instantly via local network + QR Code

Usage

  1. Install the package: npm install -g opencode-pixel-office
  2. Run the installer: opencode-pixel-office install
  3. Start OpenCode and open http://localhost:5100
